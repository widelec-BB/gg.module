NAME

    HttpGetRequest()

SYNOPSIS

    UBYTE *HttpGetRequest(STRPTR url, LONG *data_len, STRPTR user_agent)

FUNCTION

    Funkcja wysyła zapytanie GET protokołu http.

INPUTS

    - url -- adres url (bez "http://");
    - data_len -- wskaźnik na zmienną, w której zostanie umieszczona wielkość przeczytanych danych;
    - user_agent -- pozwala na zmianę pola "User-Agent" w danych http, w przypadku NULL wartość domyślna http.stream.

RESULT

    Funkcja zwraca wskaźnik na bufor z przeczytanymi danymi.

NOTES

    Bufor powinien zostać zwolniony przez użycie FreeVec()

NAME

    HttpPostRequest()

SYNOPSIS

    UBYTE *HttpPostRequest(STRPTR url, struct TagItem *postdata, LONG *data_len, STRPTR user_agent)

FUNCTION

    Funkcja wysyła zapytanie POST protokołu http.

INPUTS

    - url -- adres url (bez "http://"!);
    - postdata -- wskaźnik na taglistę z danymi do zapytania Post, taglista powinna być zgodna z http.stream;
    - data_len -- wskaźnik na zmienną, w której zostanie umieszczona wielkość przeczytanych danych;
    - user_agent -- pozwala na zmianę pola "User-Agent" w danych http, w przypadku NULL wartość domyślna http.stream.

RESULT

    Funkcja zwraca wskaźnik na bufor z przeczytanymi danymi.

NOTES

    Bufor powinien zostać zwolniony przez użycie FreeVec()
